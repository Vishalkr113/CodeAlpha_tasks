# Basic Chatbot 

## Description 

This is a simple Python-based **rule-based chatbot** developed as part of the CodeAlpha Internship.

The chatbot interacts with users through the console and responds to predefined inputs such as greetings, questions, and basic commands.

---

## Features

* Responds to greetings (hello, hi, hey)
* Answers basic questions (how are you, who are you)
* Provides simple interactions (jokes, facts, motivation)
* Supports basic calculator operations (+, -, *, /)
* Repeat user input feature
* Handles invalid inputs gracefully
* Exit command ("bye")

---

## Concepts Used

* Conditional Statements (if-elif)
* Loops (while loop)
* Functions
* String Handling
* Lists & Random Module
* User Input/Output

---

## Technologies Used

* Python (Core Programming)
