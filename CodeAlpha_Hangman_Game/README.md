# Hangman Game (Console-Based)

## Description
This is a simple text-based Hangman game built using Python.  
The player guesses a hidden word one letter at a time.

## Features
- 5 predefined words
- Random word selection
- Maximum 6 incorrect guesses
- Input validation
- Console-based interaction

## Concepts Used
- random module
- while loop
- if-else statements
- lists and strings

## How to Run
1. Install Python
2. Open terminal
3. Run:
   python hangman.py
